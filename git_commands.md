## GIT TAGS

`git tag`  
`git tag -l "v1.8.5*"`  
`git tag -a v1.2 9fceb02 -m "Message here"`  

`git push origin v1.2`  
`git push origin --tags`  

`git tag -d <tag_name>`  
`git push --delete origin tagname`  

`git checkout v2.0.0`  

`git show v1.4`  
`git show-ref --tags`  
`git fetch --tags origin`
