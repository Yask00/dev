Format document/selection  
    `Ctrl + K + F`  
    `Ctrl + Shift + I`  

Fold/Unfold code  
    `Ctrl + k + 0`  
    `Ctrl + k + j`  
    `Ctrl + Shift + ]`  
    `Ctrl + Shift + [`  


Multiple selections of the same letter/word  
    `Ctrl + d`  

Copy whole line  
    `Shift + Alt + up/down`  

Open/Close Sidebar  
    `Ctrl+ b`  

Delete last word  
    `Ctrl + Backspace`  

Select word
    `Ctrl + Shift + Left/Right`

Delete line  
    `Ctrl + x`  

Add cursor above/below  
    `Ctrl + Alt + Up arrow / Down arrow`  

Rename Symbol  
    `Select and then F2`  

Select whole line  
    `Shift + end`  
    `Ctrl + c` (without selection)  

Switch between panes  
    `Ctrl + 1/2/3`
